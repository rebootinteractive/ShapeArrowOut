# Contributed levels

Levels designed in the in-game editor land here.

Workflow:
1. In the editor, hit **↓ Download** — you get a `.json` file named after the level.
2. Move that file into this folder (`src/levels/contributed/`).
3. Commit and push to `main` — GitHub Pages redeploys automatically and the level
   appears in the menu's level list for everyone.

Files are auto-discovered at build time; no code changes needed.
